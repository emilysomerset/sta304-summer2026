JupyterHub Guide

How to access and use the course computing environment on JupyterHub.

Accessing JupyterHub
  1. Go to [ JupyterHub URL ] and log in with your UTORid.
  2. Your personal workspace will load automatically.
  3. Homework notebooks will appear in your sta304/ folder at the start of each week.
Downloading Your Notebook
  1. Open your completed notebook in JupyterHub.
  2. From the menu, choose File → Save and Export Notebook As → PDF.
  3. Your browser will download the PDF file.
  4. Submit the PDF to MarkUs.
Tips
  • Always save your work before closing the browser (Ctrl+S / Cmd+S).
  • Run all cells from top to bottom before submitting: Kernel → Restart & Run All.
  • If your server stops responding, go to File → Hub Control Panel → Stop My Server, then restart.
